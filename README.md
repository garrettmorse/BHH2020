# Additional Suffering

## Setup
If you want to check out the app, then you need Expo, Node.js, npm, etc. Clone this bad boi and rip npm ci and expo start.

## Notable Features

- Emergency contact
  - clinician
  - friend/family member
  - etc
- time/date stamp
- export/share function

- button toggle input
- symptom severity scale
- verbose text input
